Detailed Description

Implementação do algoritmo de verificação de conectividade em grafos.

O programa lê uma matriz de adjacência de um arquivo de texto (contendo 0 e 1), armazena os dados em uma estrutura estática e executa o algoritmo de busca em profundidade (DFS) de forma iterativa, sem uso de recursividade nem alocação dinâmica. O resultado indica se o grafo é ou não conexo.

Instructions:
  • Na pasta raiz do projeto haverá uma pasta 'src/validation_data' contendo arquivos de exemplo com matrizes de adjacência (0s e 1s) que representam grafos gerados com diferentes probabilidades de haver conexão entre par de vértices. Estes arquivos podem ser usados para testar o programa. OBS: O caminho definido deve ser relativo à pasta onde fica o executável.
  • Na pasta src, compile o código com: gcc ./grafo_conexo.c -o grafo_conexo
  • Execute o programa: ./grafo_conexo "caminho_do_arquivo.txt"

Lê a matriz de adjacência a partir de um arquivo de texto.

Parameters
caminhoArquivoCaminho relativo ou absoluto do arquivo a ser lido.
Returns
int Retorna o número de vértices (N) lidos com sucesso, ou -1 se houver erro de leitura.

A função realiza duas leituras: (1) Determina o número de colunas (N) lendo a primeira linha. (2) Reabre o arquivo e lê toda a matriz N x N de valores binários.

◆ verificarConectividade()

int verificarConectividade ( int n)

Verifica se o grafo é conexo utilizando busca em profundidade (DFS) iterativa.

Parameters
nNúmero de vértices (tamanho da matriz de adjacência).
Returns
int Retorna 1 se o grafo for conexo, 0 caso contrário.

A função utiliza uma pilha estática para explorar vértices conectados, evitando o uso de recursão. Cada vértice visitado é marcado no vetor global 'visitado'. Ao final, se todos os vértices forem visitados, o grafo é conexo.
